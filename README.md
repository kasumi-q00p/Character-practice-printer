# 文字練習プリンター

ひらがな、カタカナ、アルファベットの練習シートを簡単に作成・印刷できるWebアプリケーションです。

## 🌐 デモ

https://あなたのユーザー名.github.io/character-practice-printer/

## ✨ 機能

- **文字入力**: ひらがな、カタカナ、アルファベットに対応
- **改行対応**: Enterキーで改行して複数行の練習シートを作成
- **縦書き・横書き**: 書字方向を自由に切り替え
- **文字サイズ調整**: 12pt〜72ptまで7段階で調整可能
- **書き順表示**: 書き順番号の表示/非表示を切り替え
- **リアルタイムプレビュー**: 入力内容を即座にプレビュー
- **A4印刷**: ブラウザから直接印刷可能
- **レスポンシブデザイン**: PC、タブレット、スマートフォンに対応
- **アクセシビリティ**: キーボード操作、スクリーンリーダー対応

## 🚀 使い方

1. **文字を入力**: 左側のテキストエリアに練習したい文字を入力
2. **設定を調整**: 
   - 書字方向（横書き/縦書き）を選択
   - 文字サイズをスライダーで調整
   - 書き順番号の表示/非表示を選択
3. **プレビュー確認**: 右側でリアルタイムプレビューを確認
4. **印刷**: 「印刷」ボタンをクリックして練習シートを印刷

### 改行機能

名前の練習など、同じ文字を複数行書きたい場合は、Enterキーで改行できます。

例：
```
たなか はなこ
たなか はなこ
たなか はなこ
```

## 🛠️ 技術スタック

- **フロントエンド**: React 18 + TypeScript
- **スタイリング**: Styled Components
- **ビルドツール**: Vite
- **状態管理**: React Context + useReducer
- **フォント**: Google Fonts (Noto Sans JP)

## 💻 ローカル開発

### 必要な環境

- Node.js 18以上
- npm または yarn

### セットアップ

```bash
# リポジトリをクローン
git clone https://github.com/あなたのユーザー名/character-practice-printer.git
cd character-practice-printer

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで http://localhost:5173/ にアクセスしてください。

### ビルド

```bash
# プロダクションビルド
npm run build

# ビルドをプレビュー
npm run preview
```
